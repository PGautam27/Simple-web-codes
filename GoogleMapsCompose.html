Google Maps Jetpack Compose

This application displays the map and you could roll up to the place that you wanna mark on. And once you mark it up. Even after the app is killed is stores the location becaused of cashed data.

Google maps for compose came up recently at feb 2022. And there'll be much more to it in the upcoming months. We can also expect places autocomplete as a function like this Google map function.

The concepts that I used:

  • Clean architecture
  • Dagger Hilt and dependancy-Injection
  • Room Database
  • Google cloud maps sdk
  • Custom style for the maps
